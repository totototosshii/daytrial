# daytrial
## デイトラ(WP)

### 経緯
WordPress自作テーマの作成方法を学習するために制作。
HTMLファイルをPHPに書き換えるところから始め、完成後は配色やファーストビューなどをカスタマイズしました。

### ポイント
・ショートコードの使い方を学習｡<br>
・pickupタグがついた最新記事をTOPページに３件取得して表示｡<br>
・single.phpに前後記事へのページネーション設置｡<br>

### 使用言語
HTML / CSS / jQuery / PHP
